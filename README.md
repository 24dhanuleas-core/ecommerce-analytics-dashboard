# 📊 EcommerceAnalytics Dashboard

A production-ready, interactive e-commerce sales analytics dashboard built with **Streamlit**, **Plotly**, **Pandas**, and **NumPy**.

![Dashboard Preview](assets/screenshot_overview.png)

---

## ✨ Features

| Feature | Details |
|---------|---------|
| **7 Dashboard Pages** | Overview, Sales, Products, Customers, Regional, Profitability, AI Insights |
| **KPI Cards** | Revenue, Profit, Orders, Customers, AOV, Margin |
| **Global Filters** | Date range, Category, Region, State, Customer search, Product search |
| **Interactive Charts** | Area, Bar, Donut, Treemap, Choropleth, Heatmap, Scatter, Box |
| **Revenue Forecasting** | Linear trend projection with configurable horizon |
| **Auto Insights** | Rule-based business intelligence cards |
| **CSV Upload** | Bring your own dataset |
| **Data Export** | Download filtered data & customer reports |
| **Dark Theme** | Modern corporate design — Power BI / Tableau aesthetic |
